# REST API integration guide (gocertius-suite-api)

This repository ships a **Claude Code skill** that documents how to program
directly against the REST API — an alternative to using this MCP server or the
n8n node. It is a standalone, vendor-neutral guide (authentication, call
ordering, options, enums, gotchas).

**Skill file:** [`.claude/skills/gocertius-suite-api/SKILL.md`](../.claude/skills/gocertius-suite-api/SKILL.md)

Open this repo in Claude Code and the skill is available directly; or read the
file above as a reference guide.
