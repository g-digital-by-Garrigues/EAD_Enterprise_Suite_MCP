# Security Policy

## Reporting a Vulnerability

If you discover a (suspected) security vulnerability in this MCP server, please
report it privately to:

**g-digital@garrigues.com**

Please do **not** open a public GitHub issue for security matters.

When reporting, include where practical:

- A description of the vulnerability and its potential impact.
- Steps to reproduce or a proof of concept.
- Affected version(s) and deployment surface (npm, Docker image, n8n node, etc.).

We aim to acknowledge new reports within 3 business days and will keep you
informed of the remediation progress.
